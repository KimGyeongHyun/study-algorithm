재귀 함수

함수안에 본인을 호출하는 함수

보통 dfs 구현할 때 사용
말단의 값부터 계산하면서 올라와야 하는 상황에 사용 (총합)
순회 규칙만 알려주고 알아서 계산하게끔 맡김
따라서 규칙만 잘 알려준다면 직접 계산 방법을 도출하지 않고 맡길 수 있음
최적의 계산 방법을 도출하지 못한다면 차라리 재귀 함수한테 계산을 맡기는 것이
시간, 공간복잡도상 더 좋을 수 있음

매개변수를 조정하면서 재귀 수행
매개변수는 자료구조의 key(index) 값을 가져오고 계산 후 다른 key 값을 받아 재귀 수행
순환 규칙은 매개변수(index)의 변화로 결정
규칙은 자료구조에 따로 저장되지 않기 때문에 자료구조를 사용, 갱신하기 위해선
다시 매개변수(index) 규칙대로 재귀함수를 돌아 도달해야 한다

처리
1. 글로벌 변수를 재귀 함수를 돌며 갱신한다
2. 최종 도출할 값을 재귀적으로 리턴
도출할 값을 글로벌 동시에 변수에 갱신하면서 진행할 수 있음
    동일한 인덱스를 많이 접근해야 할 때 유용
    이전에 접근했던 인덱스에 접근할 때 다시 재귀적으로 다시 접근하기보단
    dp 를 사용하여 저장하는 것
    ex. 분리 집합의 find 함수 최적화
