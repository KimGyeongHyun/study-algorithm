유클리드 호제법

a와 b의 최대공약수는 b와 a%b 의 최대공약수와 같다

GCD(A, B) = GCD(B, R)

a와 b는 서로소라고 할 때 A = Ga, B = Gb 라고 하면 G 는 최대공약수
A = Bq + R (q 는 가능한 최대 몫)
R = G(a - bq)

B = Gb / R = G(a-bq) 이므로
B와 R의 최대공약수가 G 이려면
b와 a-bq가 서로소임을 증명하면 된다

b = mk / a-bq = mk' 즉, b, a-bq 가 가지는 공약수 m이 존재한다고 가정
정리하면 a = m(kq+k') / b = mk 가 되는데 a, b 는 서로소이다
따라서 b, a-bq 가 가지는 2 이상의 공약수 m 은 존재할 수 없으므로 
b, a-bq 도 서로소이다

따라서 B = Gb / R = G(a-bq) / b, a-bq 가 서로소이므로
GCD(A, B) = GCD(B, R) 가 성립한다
